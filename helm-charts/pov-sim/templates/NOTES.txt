Frontend Service:
{{- if eq .Values.frontend.service.type "LoadBalancer" }}
  NOTE: It may take a few minutes for the LoadBalancer IP to be available.

  Run the following command to get the external IP:

    kubectl get svc frontend -n {{ .Values.global.namespace }} --watch

  Once the EXTERNAL-IP is available, access the frontend at:

    http://<EXTERNAL-IP>:{{ .Values.frontend.service.port }}
{{- else if eq .Values.frontend.service.type "NodePort" }}

  Get the application URL by running:

    export NODE_PORT=$(kubectl get svc frontend -n {{ .Values.global.namespace }} -o jsonpath='{.spec.ports[0].nodePort}')
    export NODE_IP=$(kubectl get nodes -o jsonpath='{.items[0].status.addresses[0].address}')
    echo "http://$NODE_IP:$NODE_PORT"
{{- else }}

  Access the frontend using port-forward:

    kubectl port-forward -n {{ .Values.global.namespace }} svc/frontend {{ .Values.frontend.service.port }}:{{ .Values.frontend.service.port }}

  Then visit: http://localhost:{{ .Values.frontend.service.port }}
